# SomMark Playground

SomMark Playground is a web-based tool for experimenting with SomMark, a markup language for creating structured documents.

![SomMark Playground](./public/screenshots/screenshot.png)

To contribute or use this project:

1. Clone the repo
   ```
   git clone https://github.com/adam-k-smith/SomMark-Playground.git
   ```

2. Install dependencies:
   ```bash
   cd SomMark-Playground
   npm install
   ```

3. Run the development server:
   ```sh
   npm run dev
   ```
